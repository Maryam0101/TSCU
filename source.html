Where is it?

Time Series Classification Utility (TSCU) is freely available from GitHub. Majority of the code lies in just one MATLAB script: tscu.m. However it is recommended to fetch the whole repository (which is about 1Mb) in order to obtain a few dependent scripts.

You can use the following command to download the repository:

git clone https://github.com/hkayabilisim/TSCU.git

This will checkout the repository into a new directory named TSCU.
