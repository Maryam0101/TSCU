Abstract

Time Series Classification Utility (TSCU) is a collection of MATLAB and C functions written to create an easy to use framework for classification of time series. If you have a collection of time series need to be classified, then congratulations! You are in the right spot and reading the correct document.

Introduction

A time series is a collection of measurements over time. There is a large collection of time series on UCR time series repository. There are more than 40 datasets. You can send an e-mail to Dr. Keogh to download all of them. One of the datasets is available for download: Synthetic Control Dataset. If you didn't already downloaded it, then go ahead and run the following commands to fetch into the MATLAB workspace.

<mwsh:comments>%urlwrite('http://www.cs.ucr.edu/~eamonn/time_series_data/synthetic_control_TRAIN',...</mwsh:comments>
<mwsh:comments>%    'synthetic_control_TRAIN');</mwsh:comments>
<mwsh:comments>%urlwrite('http://www.cs.ucr.edu/~eamonn/time_series_data/synthetic_control_TEST',...</mwsh:comments>
<mwsh:comments>%    'synthetic_control_TEST');</mwsh:comments>
trn=load(<mwsh:strings>'synthetic_control_TRAIN'</mwsh:strings>);
tst=load(<mwsh:strings>'synthetic_control_TEST'</mwsh:strings>);

The format of the dataset is very simple. Every line corresponds to a single time series. The first column is used to store the class labels. So the length of time series is the number of columns minus 1. It will be same for testing set since the length of all time series in a repository is constant.

n = size(trn,2)-1;
fprintf(<mwsh:strings>'The length of time series..............: %d\n'</mwsh:strings>,n);

The length of time series..............: 60

You can find the number of time series in each set by counting the number of rows as follows.

trnsize = size(trn,1);
tstsize = size(tst,1);
fprintf(<mwsh:strings>'The number of time series in traning...: %d\n'</mwsh:strings>,trnsize);
fprintf(<mwsh:strings>'The number of time series in testing...: %d\n'</mwsh:strings>,tstsize);

The number of time series in traning...: 300
The number of time series in testing...: 300

Now lets find out the class labels. As I said before, the class labels are in the first column. You can determine the class labels by using unique.
