Examples

  • First of all, please download synthetic control dataset from UCR time series repository. It is provided as free. You can also request the rest of the dataset from Eamonn Keogh.
  • Assuming you downloadaed the training and testing set, you should first load the dataset as follows:
    trn=load('synthetic_control_TRAIN'); tst=load('synthetic_control_TEST');
    You can then classify the time series in the testing set by using the default options as follows:
    tscu(trn,tst)

  • If you want to use Dynamic Time Warping then you should append 'alignment' option as follows:
    tscu(trn,tst,'alignment','DTW')
  • You can also use Constrained Dynamic Time Warping by using 'CDTW' option as follows:
    tscu(trn,tst,'alignment','CDTW')
    In this case, band width is taken as 6% by default.
  • If you want to change also the band width than you can use 'DTWbandwidth' option:
    tscu(trn,tst,'alignment','CDTW','DTWbandwidth',10)
